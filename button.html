Button

Solid Button

The solid button component is available in three variants btn-primary ,btn-secondaryandbtn-danger. 

          
          <div class="btn-btn">
            <button class="btn-links btn-primary">Primary</button>
            <button class="btn-links btn-secondary">Secondary</button>
            <button class="btn-links btn-danger">Danger</button>
          </div>

Outline Buttons

The button component is available in three variants btn-outline-primary ,btn-outline-secondaryandbtn-outline-danger. 

          
        <div class="btn-btn">
          <button class="btn-links btn-outline-primary">Primary</button>
          <button class="btn-links btn-outline-secondary">Secondary</button>
          <button class="btn-links btn-outline-danger">Danger</button>
        </div>

Icon Buttons

Add the btn-icon class to the icon element with the link of the icon. 

          
           <div class="btn-btn">
            <button class="btn-links btn-primary">
             <ion-icon class="icon" name="cart-outline"></ion-icon>Primary
            </button>
            <button class="btn-links btn-danger">
              <ion-icon class="icon" name="trash-outline"></ion-icon>Danger
            </button>
          </div>
